CNN Image Classifier with Flask

This project is a simple web application built with Flask that allows you to upload an image and classify its contents using pre-trained Convolutional Neural Networks (CNNs) from the torchvision library. It supports switching between different models, such as ResNet18 and VGG16, for image classification and provides the ability to download the classification results.
